Plugin: Sort Rows

Description

The Sort Rows transform sorts rows based on the fields you specify and on whether they should be sorted in ascending or descending order.
The transform optionally only pass unique records, based on the sort keys.
 | You use can use multiple copies of the Sort Rows transform to speed up large sort operations. Make sure to add aSorted Mergetransform after the sort to correctly merge the streams of sorted that are generated by the various sort copies.
