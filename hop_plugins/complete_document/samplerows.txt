Plugin: Sample Rows

Description

The Sample Rows transform retains a sample set of rows.


Options

Option | Description
Transform name | the name for this transform
Lines range | Specify a comma separated list of rows you want to sample. This list can contain single line numbers or line number ranges, e.g10,20,30,40,50..60to sample rows 10, 20, 30, and 40, and all lines from 50 to 60.
Line nr fieldname | fieldname to store the line number
